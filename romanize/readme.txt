---Файлы---
1)Файл romanize.py содержит функцию romanize, которая позволяет переводить арабские числа в римские.
2)Файл test_launch.py содержит цикл для проверки вышеуказанной функции, чтобы выйти из цикла необходимо ввести -1
---О romanize---
romanize принимет в себя значения от 1 до 399 (поскольку в коде задачи максимальным числом цикла
была римская "C" (100), а максимальное число доступное без использования "D" (500) это 399, т.к. 400 == "CD").
В случае попытки ввести значения не входящий в этот диапазон будет выдана ошибка 'Number should be in [1, 399] range'
